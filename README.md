# CitationLake
## Features
- Extract text, figures, and tables from PDFs and links.
- Build and manage citation graphs.
- Define and execute tasks such as model retrieving, table integration, and data lake operations.
- User-friendly UI with pipeline visualization and task management.# CitationLake

```markdown
# Project Directory Structure
├── data                     # Folder for storing raw and processed data.
├── examples                 # Example datasets or files to demonstrate functionality.
├── src                      # Source code folder containing main project modules.
│   ├── extractors           # Code for extracting text, tables, and figures from PDFs and links.
│   ├── graph                # Code for creating, managing, and visualizing citation graphs.
│   ├── pipeline             # Code for building and executing task pipelines.
│   ├── tasks                # Code for defining and executing specific tasks (e.g., model retrieving).
│   ├── ui                   # Code for user interface components and templates.
│   └── utils                # Utility functions for logging, configuration, and file handling.
└── tests                    # Unit tests and integration tests for project modules.
```
